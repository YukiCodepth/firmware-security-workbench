# Sample Corpus

This corpus provides safe text-based firmware-like files for demos and tests.

Files:

- `samples/corpus/esp32-lab-vuln.bin`
- `samples/corpus/stm32-lab-vuln.bin`
- `samples/corpus/arduino-lab-vuln.bin`
- `samples/corpus/pico2-lab-vuln.bin`

Purpose:

- validate secret detection
- validate endpoint detection
- validate SBOM candidate extraction
- validate CVE candidate matching
- demo firmware diff and report export

Safety note:

These are synthetic training/demo files. They are not extracted from real products.
