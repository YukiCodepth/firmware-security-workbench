# Phase 0: Project Foundation

## Goal

Create the project repository foundation before writing scanner code.

## What You Should Learn In This Phase

- what a Git repository is
- why `main` should stay stable
- why each phase gets a separate branch
- how open-source projects are structured
- why documentation matters before the codebase grows

## Commands Used

```bash
git init -b main
git add .
git commit -m "Initialize repository"
git checkout -b phase/00-project-foundation
```

## Phase Completion Checklist

- repo initialized
- `main` branch created
- phase branch created
- project structure added
- roadmap added
- branching strategy documented
- phase branch committed
- phase branch merged into `main`
